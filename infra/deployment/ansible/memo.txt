# 테스트 확인
이 파일과 같은 레벨에 hosts.ini 파일 생성

[onprem_db]
<IP>

[ccmall_rec]
<IP> ansible_user=ec2-user ansible_ssh_private_key_file=./tailscale-key.pem

[ccmall_web]
<IP> ansible_user=ec2-user ansible_ssh_private_key_file=./tailscale-key.pem

[ccmall_con]
<IP>

IP 입력 후 터미널을 현재 폴더로 change
-> ping 확인 후
-> ansible-playbook test_setup.yml 하시면 앤서블은 돌아가는 걸 확인하실 수 있습니다.
